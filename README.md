# Smart Safety Alert System

A real-time safety monitoring system using YOLOv8 and OpenCV.

## Description
This project detects humans entering a defined danger zone using live camera input.
When a person enters the danger zone, an alert sound is triggered to warn nearby people.

## Technologies Used
- Python
- OpenCV
- YOLOv8
- Computer Vision

## Use Case
- Workplace safety
- Restricted area monitoring
- Smart surveillance systems
